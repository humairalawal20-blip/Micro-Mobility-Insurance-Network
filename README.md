# Micro-Mobility Insurance Network

## Overview

The Micro-Mobility Insurance Network is a revolutionary blockchain-based insurance system designed specifically for electric scooters, bikes, and other micro-mobility devices. This system provides pay-per-use insurance with real-time risk assessment, making micro-mobility more accessible and affordable for everyone.

## 🚀 Key Features

### Pay-Per-Use Insurance
- **Dynamic Pricing**: Insurance premiums calculated based on actual usage
- **Real-Time Assessment**: Instant risk evaluation using telematics data
- **Flexible Coverage**: Insurance that scales with your mobility needs

### Smart Risk Management
- **Route Risk Analysis**: Premium adjustments based on route safety metrics
- **Weather Integration**: Real-time weather condition assessments
- **Rider Behavior Monitoring**: Personalized risk profiles based on riding patterns

### Blockchain Benefits
- **Transparency**: All transactions and risk assessments on-chain
- **Automation**: Smart contracts handle policy execution and claims
- **Cost Efficiency**: Reduced overhead through decentralized operations

## 🏗️ System Architecture

The system consists of two main smart contracts:

### 1. Vehicle Telematics Oracle
- Collects real-time data from micro-mobility devices
- Tracks usage patterns, location, and device health
- Provides secure data feeds for risk assessment

### 2. Dynamic Premium Calculator
- Processes telematics data for risk analysis
- Calculates dynamic insurance premiums
- Integrates weather and route risk factors

## 🛡️ Insurance Coverage

### Device Protection
- **Theft Coverage**: Protection against device theft
- **Damage Coverage**: Repair costs for accidental damage
- **Third-Party Liability**: Coverage for injuries or property damage

### Usage-Based Pricing
- **Distance-Based**: Pay for the miles you ride
- **Time-Based**: Hourly or daily coverage options
- **Risk-Adjusted**: Lower premiums for safer routes and behavior

## 🔧 Getting Started

### Prerequisites
- Clarinet CLI tool
- Node.js and npm
- Git

### Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Run tests: `clarinet test`
4. Deploy contracts: `clarinet deploy`
